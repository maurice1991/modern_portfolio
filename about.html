BIO

Als eerste welkom op mijn portfolio! Graag wil ik mij even voorstellen en u laten zien wat ik graag doe. Ik ben Maurice van Ooijen, 28 jaar en ben woon achtend in Zetten, dit is een dorpje in het mooie Overbetuwe. Ik ben geboren in een dorp naast zetten genaamd Herveld-Andelst, waar ik ben opgegroeid met twee ouders en mijn zusje. Mijn grootste hobby’s zijn kickboksen, programmeren, Netflixen, game en leuke dingen doen met vrienden of familie. Momenteel heb ik geen baan en zou graag weer iets doen in de IT en dan vooral op het vlak van programmeren van standalone applicaties, web applicaties of mobile apps.
Waarom?
Omdat ik gewoon een passie heb voor het ‘waar’ maken van ideeën binnen de business en hier graag elke dag in zou willen door groeien. Dit omtrent alle processen binnen het programmeren, zowel de architectuur, het ontwikkelen als het testen en implementeren. Ik heb een uitputtende honger naar kennis binnen het programmeer vak en heb hier ook al een baan in gehad. Heb ik uw interesse? Neem dan contact met mij op!

HTML5, CSS3 en SASS en JS

Webdesigner

Voor het maken van websites maak ik gebruik van de nieuwste technologieën. Deze website is opgebouwd d.m.v. HTML5, CSS3, SASS en JavaScript. Ik vind het leuk om dingen van scratch op te bouwen en heb daarom ook geen gebruik gemaakt Wordpress of e.d. CMS. Tevens heb ik dan ook overzichtelijk waar de fout ligt, wanneer ik een bug of error tegenkom op mijn website(duh dat doe je editor VisualCode). Voor versioncontrol gebruik ik Github wat naar mijn idee een onmisbare tool is bij het bouwen van applicaties en websites.

Webapplicaties

Front end Developer, React JS..

Voor het maken van webaplicaties maak ik gebruik van de door facebook gemaakte Java libary/framework React JS. Het gebruik van react geeft een efficiënte en overzichtelijke manier van coderen omdat het gebruik maakt van componenten. React maakt gebruik van een zogenaamde 'virtual DOM', dit wilt zeggen dat het beheren en updaten van elementen in de pagina een stuk eenvoudiger wordt.
